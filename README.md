# AddToPath

AddToPath is a simple program that adds a directory to the PATH environment variable.

## Usage

### Add

```shell
AddToPath.exe <directory>
AddToPath.exe add <directory>
```

### Check

```shell
AddToPath.exe check
```

### List
    
```shell
AddToPath.exe list
```

### Help

```shell
AddToPath.exe -h | --help
AddToPath.exe -v | --version
```
